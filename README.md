twitter-chess
=============

Play Chess using Twitter. For profit and epic lulz.

Requirements

Requirements
------------

* Node.js
* Grunt CLI:
```
npm install -g grunt-cli
```
* Bower:
```
npm install -g bower
```

After Checkout
--------------

```
npm install
bower install
```

Usage
-----
* run the server
```
grunt
```

* runt tests
```
grunt test
```
